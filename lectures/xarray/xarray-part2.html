Lesson 2: Advanced Xarray

In this lesson, we cover some more advanced aspects of xarray.

Groupby

Xarray copies Pandas‚Äô very useful groupby functionality, enabling the ‚Äúsplit / apply / combine‚Äù workflow on xarray DataArrays and Datasets. In the first part of the lesson, we will learn to use groupby by analyzing sea-surface temperature data.

import numpy as np
import xarray as xr
from matplotlib import pyplot as plt

First we load a dataset. We will use the NOAA Extended Reconstructed Sea Surface Temperature (ERSST) v5 product, a widely used and trusted gridded compilation of of historical data going back to 1854.

As we can see from the plot, the timeseries at any one point is totally dominated by the seasonal cycle. We would like to remove this seasonal cycle (called the ‚Äúclimatology‚Äù) in order to better see the long-term variaitions in temperature. We will accomplish this using groupby.

The syntax of Xarray‚Äôs groupby is almost identical to Pandas.

Apply & Combine

Now that we have groups defined, it‚Äôs time to ‚Äúapply‚Äù a calculation to the group. Like in Pandas, these calculations can either be:

  • aggregation: reduces the size of the group
  • transformation: preserves the group‚Äôs full size

At then end of the apply step, xarray will automatically combine the aggregated / transformed groups back into a single object.

The most fundamental way to apply is with the .apply method.

gb.apply?

Apply uses heuristics (like `pandas.GroupBy.apply`) to figure out how

1. If the dimension along which the group coordinate is defined is

   over this dimension.
2. Otherwise, stack over the new dimension given by name of this

Transformations

Now we want to remove this climatology from the dataset, to examine the residual, called the anomaly, which is the interesting part from a climate perspective. Removing the seasonal climatology is a perfect example of a transformation: it operates over a group, but doesn‚Äôt change the size of the dataset. Here is one way to code it

def remove_time_mean(x):
    return x - x.mean(dim='time')

ds_anom = ds.groupby('time.month').apply(remove_time_mean)
